# 说明

这个项目目的是使用不同语言实现一些科学计算功能等，如（偏）微分方程数值解、曲线拟合等。预计会有中文(zh-CN)和英文(en-US)两种语言的注释版本（仅注释的语言不同，代码一致）。
不同的语言及其预实现的功能如下：

> [!NOTE]
> 下文所提到的语言标准/版本均为我使用的标准/版本，不代表运行项目所要求的最低/高版本。
