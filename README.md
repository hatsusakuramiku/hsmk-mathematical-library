# 说明

这个项目目的是使用不同语言实现一些科学计算功能等，如（偏）微分方程数值解、曲线拟合等。

> [!NOTE]
> 下文所提到的语言标准/版本均为我使用的标准/版本，不代表运行项目所要求的最低/高版本。

## C （C 语言）

> **语言标准**: C11
> **编译器**: gcc.exe (x86_64-posix-sjlj-rev0, Built by MinGW-W64 project) 8.1.0
> **文件编码**: UTF-8
> **字符集**: UTF-8
