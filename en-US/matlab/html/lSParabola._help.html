lSParabola

使用最小二乘法进行曲线拟合

语法

[polynomial, coefficients] = lSParabola(xCoordinate, yCoordinate)

说明

[polynomial, coefficients] = lSParabola(xCoordinate, yCoordinate)
返回基于给定的坐标数据 (xCoordinate, yCoordinate) 使用最小二乘法对特定曲线形式进行曲线拟合的结果。
  • 请确保进行拟合的坐标数据维度与长度均相等。
  • 请确保进行拟合的 xCoordinate 中元素各不相等。

示例

拟合形如 的曲线
fun = @(x) x .^ 2 + 2 .* x + 3;
xCoordinate = -5:0.5:5;
xLine = linspace(-5,5);
yCoordinate = fun(xCoordinate);
[polynomial, ~] = lSParabola(xCoordinate, yCoordinate);
plot(xCoordinate, yCoordinate, 'r*', xLine, polynomial(xLine), 'b-');
xlabel("x");ylabel("y");

输入参数

xCoordiate - x坐标
要求类型：向量
支持数据类型：double
参数要求：各元素应互不相等
yCoordinate - y坐标
要求类型：向量
支持数据类型：double

输出参数

polynomial - 最终拟合得到的表达式
类型：标量
数据类型： function_handle
说明：输出内容是关于 x 的二次函数。
coefficients - 最终拟合得到的表达式的系数
类型：向量
数据类型：double
说明：向量的第一个元素是列出的曲线形式中的系数 C 的值；向量的第二个元素是列出的曲线形式中的系数 B 的值；向量的第二个元素是列出的曲线形式中的系数 A 的值。
